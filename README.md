# Student Management System (C Language)

## 📌 Overview
This project is part of my C programming learning journey.

I am learning C step-by-step and improving the **same project** as I master new concepts.  
Instead of waiting to learn everything first, I am building and upgrading this system version by version.

Current Version: **V1**

---

## 🚀 Features (V1)
- Add students
- Display students
- Search student
- Update student
- Delete student
- Menu-driven console program

This version stores student names using arrays and functions.

---

## 🧠 Concepts Used
- Basic C Program Structure
- Variables & Data Types
- Operators
- Conditional Statements
- Loops
- Functions
- Pointers
- Arrays & Multidimensional Arrays
- String Handling
- Modular Programming

---

## 🏗️ Project Philosophy
This repository represents **learning in public**.

Each new concept learned in C will upgrade this same project:

- V1 → Arrays based system  
- V2 → Structured student records  
- V3 → Persistent storage using files  

The goal is to observe how a program evolves as programming knowledge grows.

## 🛠️ How to Run

### Compile
```bash
gcc main.c -o sms
```

### Run
```bash
./sms
```

*(Windows users can run `sms.exe`)*
